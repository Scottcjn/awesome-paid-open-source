# Contributing to awesome-paid-open-source

Thank you for your interest in contributing! This list curates paid open source opportunities and we welcome quality additions.

## How to Contribute

### Adding a New Entry

1. **Fork the repository** and clone it locally
2. **Add your entry** to the appropriate section in `README.md`
3. **Follow the format**: `- [Project Name](link) - Brief description`
4. **Ensure the entry is valid**: The project must offer paid contributions (bounties, sponsorships, grants, etc.)
5. **Submit a Pull Request** with a clear description

### Guidelines

- **Quality over quantity**: Only add projects with active paid contribution opportunities
- **Verify links**: Ensure all URLs are working and point to relevant pages
- **Be descriptive**: Brief descriptions should clearly state the payment model
- **Alphabetical order**: Keep lists alphabetically sorted within sections
- **No duplicates**: Check that the project isn't already listed

### Pull Request Checklist

- [ ] Entry follows the existing format
- [ ] Link is valid and points to bounty/contribution page
- [ ] Description is concise and mentions payment type
- [ ] Added to the correct section
- [ ] No duplicate entries

## Code of Conduct

- Be respectful and inclusive
- Focus on constructive feedback
- Help maintain list quality

## Questions?

Open an issue for discussions or clarifications.
